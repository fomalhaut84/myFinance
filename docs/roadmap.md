# Roadmap

- [x] Next.js 14 + TypeScript + Tailwind 프로젝트 초기화
- [x] Prisma 설정 + PostgreSQL + schema.prisma 적용
- [x] 시드 스크립트 작성 (3계좌 보유종목, `@docs/seed-data.md` 참조)
