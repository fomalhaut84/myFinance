## 15-A: 아이 전용 간소화 대시보드

### 요구사항

- [ ] /kids/[accountId] 페이지
- [ ] 나이 기반 레벨 자동 판정 (5~8: 씨앗, 9~12: 새싹, 13~15: 나무)
- [ ] 씨앗: 총 금액 + 성장률 + 종목별 한줄 설명 + 이모지
- [ ] 새싹: 종목별 수익률 + 파이차트 + 배당금 안내
- [ ] 큰 글씨, 이모지, 시각적 디자인

