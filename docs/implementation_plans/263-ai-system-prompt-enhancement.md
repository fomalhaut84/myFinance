# 구현 계획: AI 시스템 안내 강화 (#263)

## 변경 파일 (1개)

`src/lib/ai/system-prompt.ts` — SYSTEM_PROMPT 4개 섹션 추가

### 추가 섹션

1. **텔레그램 봇 커맨드 (사용자 안내용)**
   - `관심 [종목] 매수구간 [하한] [상한]`
   - `전략 [종목] 목표가 [가격]` / `손절 [가격]` / `매수구간 [하한] [상한]`
   - `매수/매도 [계좌] [종목] [수량] [가격]`
   - 자연어 거래/가계부 입력
   - `/관심목록`, `/현황`, `/계좌`, `/주가` 등

2. **알림 시스템 현황**
   - 급등락 알림 (장중만)
   - 목표가/손절가 도달, 매수구간 진입
   - TA 시그널 (스윙/모멘텀/단타 종목)
   - 일일 요약 (매일 08:00 KST, D-1 변동 포함)
   - 모닝 브리핑 (한국장/미국장)
   - 분기 점검 + 분기 리포트 PDF
   - RSU/스톡옵션 베스팅 리마인더

3. **시스템 정보 안내 규칙**
   - CLAUDE.md / docs 디렉토리 직접 참조 금지
   - 기능 가능 여부 = 시스템 프롬프트 + MCP 도구 목록 + 위 봇 커맨드 안내 기반으로만 판단
   - "구현 안 됨"이라고 답하기 전에 시스템 프롬프트 다시 확인

4. **쓰기 도구 사용 규칙** (Phase 23-24 대비, 미리 정의)
   - 조회 도구(`get_*`): 즉시 실행
   - 추가/수정 도구(`add_*`, `update_*`, `set_*`): 사용자에게 "다음과 같이 등록/수정할까요?" 요약 보여주고 확인 후 실행
   - 삭제 도구(`delete_*`): 사용자 명시적 동의 필요 ("삭제할까요?" 후 yes 응답)
   - 호출 결과는 사용자에게 한국어로 요약
