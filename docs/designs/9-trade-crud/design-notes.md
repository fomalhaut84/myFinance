# Phase 3 Trade CRUD — 디자인 노트

## 디자인 방향

기존 Phase 1~2 디자인 시스템 100% 유지. 새로운 디자인 요소 최소화, 기존 패턴 재사용.

## 신규 페이지

### 1. 거래 내역 (/trades)
- Header + 필터 바 + 테이블 + 페이지네이션
- 필터: 계좌 (전체/세진/소담/다솜) + 유형 (전체/매수/매도)
- 테이블: 기존 HoldingsTable 스타일 재사용
- BUY 뱃지: `bg-sejin/10 + text-sejin`, SELL 뱃지: `bg-red-500/10 + text-red-500`
- 액션 버튼: 수정(펜 아이콘) + 삭제(휴지통 아이콘)

### 2. 새 거래 (/trades/new)
- 카드 내 폼 레이아웃 (max-width 560px)
- 계좌 선택: 3-column 버튼 (선택 시 계좌 컬러 border 강조)
- 유형 토글: BUY(초록) / SELL(빨강) 양분 버튼
- 종목 선택: select 드롭다운 (보유종목 + 직접입력)
- 실시간 총액 계산: 수량 × 단가 × 환율
- 제출 버튼: BUY=초록, SELL=빨강

### 3. 거래 수정 (Slide Panel)
- 우측 슬라이드 패널 (420px)
- 계좌/종목/유형은 고정 표시 (수정 불가)
- 수량, 단가, 환율, 거래일, 메모만 수정 가능

### 4. 삭제 확인 (Modal)
- 중앙 모달 + backdrop blur
- 거래 상세 정보 표시 (종목, 유형, 수량, 거래일)
- 삭제 버튼: 빨강 계열

## UI 컴포넌트 결정

| 요소 | 구현 방식 |
|------|----------|
| 필터 | SegmentedControl (기존 없음, 신규) |
| 테이블 | 기존 HoldingsTable 패턴 재사용 |
| 폼 입력 | input bg-white/[0.035], border-white/[0.06] |
| 계좌 선택 | 3-button grid, 선택 시 border 강조 |
| 유형 토글 | 2-segment toggle |
| 수정 | 우측 슬라이드 패널 |
| 삭제 | 중앙 모달 |
| 페이지네이션 | 숫자 버튼 + 이전/다음 |

## 사이드바 변경

기존 메뉴에 "거래" 항목 추가:
- 아이콘: 사각형 + 플러스 조합
- 위치: "대시보드" 아래

## 색상 참조

- 입력 필드 배경: `rgba(255,255,255,0.035)`
- 입력 필드 포커스: `rgba(255,255,255,0.055)` + border `rgba(255,255,255,0.14)`
- 필터 활성: `rgba(255,255,255,0.07)`
- 테이블 호버: `rgba(255,255,255,0.02)`
