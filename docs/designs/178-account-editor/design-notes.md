# 계좌 관리 + 설정 페이지 디자인 노트 — #178

## 디자인 방향

/settings 통합 페이지의 첫 번째 탭. 계좌 정보를 인라인 편집으로 간편하게 수정.
17-E에서 나머지 탭(알림, 근로소득, 후잉 연동) 구현 예정.

## 설정 페이지 구조

### /settings 페이지
- 탭 바: 계좌 | 알림 | 근로소득 | 후잉 연동
- 17-A에서는 "계좌" 탭만 구현, 나머지는 placeholder 또는 빈 탭

### 계좌 탭
- 3개 계좌 카드 (세진, 소담, 다솜) 세로 배치
- 각 카드: 읽기 모드 / 편집 모드 전환

## 읽기 모드

- 헤더: 컬러 도트 + 이름 (계좌별 고유 색상) + "수정" 버튼
- 본문: label:value 쌍 (전략, 투자 기간, 벤치마크, 나이)
- 미설정 값: 회색 이탤릭 "미설정"
- 컴팩트 카드 (최소 높이)

## 편집 모드

- "수정" 클릭 → 카드가 편집 폼으로 전환 (인라인)
- 카드 테두리: sodam 색상으로 강조 + "수정 중" 텍스트
- 필드: 이름, 전략 (text), 투자 기간 (number + "년"), 벤치마크 (text), 나이 (number + "세")
- 저장 → PATCH /api/accounts/[id]
- 취소 → 원래 읽기 모드로 복원
- 다른 카드는 읽기 모드 유지 (한 번에 하나만 편집)

## UI 컴포넌트

| 요소 | 구현 방식 |
|------|----------|
| 설정 페이지 | /settings 경로, 탭 컴포넌트 |
| 탭 바 | 세그먼트 컨트롤 (4칸) |
| 계좌 카드 | 읽기/편집 모드 전환 |
| 편집 폼 | 인라인 (카드 내부) |
| 저장/취소 | sodam 버튼 / border 버튼 |
