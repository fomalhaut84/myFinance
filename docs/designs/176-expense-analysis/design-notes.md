# 가계부 분석 강화 디자인 노트 — #176

## 디자인 방향

기존 가계부 페이지(/expenses)에 분석 섹션을 추가. 기존 요약 카드/차트/테이블은 그대로 유지하고 아래에 새 컴포넌트를 배치.

## 추가되는 컴포넌트

### 1. ExpenseSummary 확장 — 전월 대비 뱃지
- 기존 4개 카드(소비, 수입, 순수익, 건수) 하단에 소형 비교 뱃지
- 소비 증가: ▲ 빨강 (부정적)
- 소비 감소: ▼ 초록 (긍정적)
- 수입 증가: ▲ 초록 (긍정적)
- 순수익 감소: ▼ 빨강 (부정적)
- 변동 없음: — 회색
- 월 미선택 (연간) 시 전월 대비 숨김

### 2. MonthCompare — 전월 대비 그룹별 상세
- 테이블: 그룹 | 이번 달 | 전월 | 증감 | 증감률
- CategoryGroup 기준 합산 비교
- 증감 절대값 내림차순 정렬
- 색상: 증가=빨강, 감소=초록
- 이상 지출: 150%+ 증가 시 강조

### 3. SpendingTrend — 지출 트렌드 차트/테이블
- Recharts LineChart: 그룹별 라인 (상위 4~5개)
- 기간 선택: 3개월 / 6개월 토글
- 이상 지출: 평균 대비 150% 이상 → 빨간 테두리 원
- 차트/테이블 토글: 같은 데이터를 숫자로도 조회 가능
- 테이블: 그룹 | 월별 금액 | 평균 | 이번 달 vs 평균

## 데이터 소스
- 전월 대비: 해당 월 + 전월의 카테고리별 소비 aggregate → 그룹별 합산
- 트렌드: 최근 3~6개월 카테고리별 소비 aggregate → 그룹별 합산 + 평균 계산
- 기존 GET /api/transactions 응답 확장 또는 별도 분석 API

## 레이아웃 (가계부 페이지 내)
```
[요약 카드 (전월 대비 뱃지 추가)]     ← Scene 1
[월별 바차트] [카테고리/그룹 파이차트] ← 기존
[전월 대비 그룹별 변동]                ← Scene 2 (신규)
[지출 트렌드 차트/테이블]              ← Scene 3+4 (신규)
[내역 테이블]                          ← 기존
```
