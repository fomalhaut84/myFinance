# 네비게이션 개선 디자인 노트 — #170

## 디자인 결정사항

### 사이드바 그루핑
- 기존 플랫 리스트 15개 → 3개 섹션으로 그루핑
- 섹션 헤더: 기존 "계좌" 레이블과 동일한 스타일 (10px, uppercase, dim, tracking)
- 접기/펼치기: ▾ 화살표, 접히면 -90deg 회전
- 접기 애니메이션: max-height + opacity 트랜지션 (0.25s)
- 계좌 섹션: 접기 없음 (항상 표시)

### 모바일 더보기
- 기존 10개 → 12개 (가계부, 카테고리 추가)
- 4x3 그리드 유지 (12개가 딱 맞음)
- max-height 70vh + overflow-y-auto (항목 증가 대비)

### 교육 뷰 진입점
- 위치: 계좌 상세 페이지의 태그 행 우측 (ml-auto)
- 스타일: "🎒 교육 뷰 →" 버튼, bg-surface + border로 태그보다 약간 강조
- 조건: ownerAge != null인 계좌에서만 렌더링
- 이동: /kids/[accountId]

### 유지된 것
- 기존 다크 테마 CSS 변수 전부 유지
- Pretendard 폰트
- nav-link 활성 스타일 (bg-surface + border-border)
- 계좌 컬러 도트 (sejin/sodam/dasom)
- 하단 고정 4개 탭 (대시보드, 거래, 세금, 더보기)
