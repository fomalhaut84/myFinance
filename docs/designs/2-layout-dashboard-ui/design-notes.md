# 디자인 결정사항 — Issue #2 레이아웃 + 대시보드 UI

## 디자인 방향

**톤**: 다크 미니멀리즘 — 깊은 어두운 배경 위에 반투명 카드, 계좌별 액센트 컬러로 구분.

## 컬러 팔레트

| 용도 | 변수 | 값 |
|------|------|----|
| 배경 | `--bg` | `#07080c` |
| 카드 배경 (raised) | `--bg-raised` | `#0d0e14` |
| 카드 | `--card` | `rgba(255,255,255,0.025)` |
| 카드 호버 | `--card-hover` | `rgba(255,255,255,0.045)` |
| 보더 | `--border` | `rgba(255,255,255,0.06)` |
| 보조 텍스트 (dim) | `--dim` | `#6e6e82` |
| 보조 텍스트 (sub) | `--sub` | `#9494a8` |
| 본문 텍스트 | `--text` | `#c8c8d4` |
| 강조 텍스트 | `--bright` | `#eeeef2` |
| 세진 | `--sejin` | `#34d399` |
| 소담 | `--sodam` | `#60a5fa` |
| 다솜 | `--dasom` | `#fb923c` |

## 폰트

- **Primary**: Pretendard (한글/영문 모두)
- **Fallback**: -apple-system, BlinkMacSystemFont, system-ui, sans-serif
- **최소 폰트**: 11px (고해상도에서 12px)
- **HiDPI**: `@media (min-resolution: 144dpi)` 쿼리로 1px 범프

## 레이아웃

- **데스크톱**: 왼쪽 사이드바 220px + 메인 콘텐츠
- **모바일 (≤768px)**: 사이드바 숨김 → 하단 탭 바
- **태블릿 (769~1024px)**: 계좌 카드 2열
- **최대폭**: 콘텐츠 영역 960px

## 카드 스타일

- 반투명 배경 + 미세한 보더
- 상단 1px glow 라인 (계좌 컬러 그라데이션)
- 호버 시: 배경 밝아짐 + 보더 강화 + translateY(-1px)
- 계좌별 box-shadow glow 효과

## 차트

- **형태**: SVG 도넛 차트 (stroke-dasharray 기반)
- **크기**: 180x180px, viewBox `-4 -4 128 128` (클리핑 방지)
- **중앙**: 총 매입금 요약 표시
- **범례**: 차트 아래 세로 목록 (이름 + 비중%)

## 피드백 반영 이력

1. 도넛 차트 상하좌우 잘림 → viewBox 여백, overflow: visible
2. 배경/글꼴 명도차 부족 → --dim, --sub 색상 밝기 상향
3. 10px 폰트 고해상도 가독성 → 최소 11px + HiDPI 미디어쿼리
