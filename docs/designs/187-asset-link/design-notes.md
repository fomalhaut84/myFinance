# 가계부 자산 연동 디자인 노트 — #187

## 디자인 방향

기존 TransactionForm/TransactionTable을 확장. 유형 선택 세그먼트와 자산 연결 필드를 추가.

## 유형 세그먼트 컨트롤

| 유형 | 색상 | 자산 필드 |
|------|------|----------|
| 소비 | 빨강 배경 | 숨김 |
| 수입 | 초록 배경 | 숨김 |
| 출금 | sodam(파랑) 배경 | 표시 (파란 테두리) |
| 입금 | teal(청록) 배경 | 표시 (teal 테두리) |

## 자산 선택 필드

- 출금/입금 선택 시에만 표시
- select 드롭다운: 자산명 + [카테고리] + 현재 금액
- 임팩트 미리보기: 현재값 → 변경후값 (차이)
- 출금: 자산 감소 (초록색 차이), 입금: 자산 증가 (파란색 차이)

## 테이블 표시

- 일반 소비/수입: 기존과 동일
- transfer_out: 내용 옆에 "출금 → 자산명" 소형 뱃지 (sodam 컬러)
- transfer_in: 내용 옆에 "입금 → 자산명" 소형 뱃지 (teal 컬러)
- 금액 색상: 출금/입금 모두 빨강 (지출로 분류)

## 색상 참조

- 출금 관련: sodam #60a5fa (기존 파란색 계열)
- 입금 관련: teal #2dd4bf (청록, 출금과 시각적 구분)
- 소비: red #f87171
- 수입: emerald #34d399
