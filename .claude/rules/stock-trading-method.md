# 주식 매매 방법론 스킬

이 스킬은 유동성과 변동성 지표 기반의 실전 트레이딩 프레임워크를 체크리스트 형식으로 제공한다.
기업 분석 없이 순수 지표만으로 판단하는 전략이며, **손절 없음** 원칙을 전제로 한다.

사용자가 "살만한 종목 추천해줘", "거래량 많은 종목 뭐 있어?", "이 섹터에서 유동성 좋은 종목 찾아줘" 같이 종목 발굴을 요청하거나,
특정 종목의 매수 타이밍·매도가 설정·하락장 대응 방법을 묻거나,
"지금 들어가도 될까?", "매수가 어떻게 잡아야 해?", "이 종목 어때?" 같은 질문을 할 때 반드시 이 스킬을 사용한다.
코인은 대상 외.

---

## 출력 형식 원칙

- 각 단계를 체크리스트 형태로 제시
- 사용자가 제공한 데이터가 있으면 바로 계산해서 수치 제시
- 데이터가 없으면 "확인 필요" 항목으로 표시
- 단계별로 Pass / 확인필요 / 주의 중 하나로 판정

---

## 0단계 — 종목 발굴

> 이미 관심 종목이 있으면 이 단계는 건너뛰고 1단계로 이동.
> "살만한 종목 추천해줘", "거래량 많은 종목 찾아줘" 류의 요청일 때 실행.

### 사용자 입력 수집

먼저 아래 조건을 사용자에게 확인한다. 답변 없으면 기본값으로 진행.

| 조건 | 사용자 입력 | 기본값 |
|------|------------|--------|
| 섹터/테마 | (입력 받기) | 전체 |
| 주가 범위 | (입력 받기) | 제한 없음 |
| 변동성 수준 | 안정형 / 보통 / 공격형 | 보통 |
| 조회 방식 | 섹터 추천 / 거래량 상위 / 둘 다 | 둘 다 |

### 방식 A — 거래량 상위 종목 조회

web_search를 사용해 당일 또는 최근 거래일 기준 미국 주식 거래량 상위 종목을 검색한다.

### 방식 B — 섹터/테마 기반 추천

사용자가 섹터 또는 테마를 입력한 경우, 해당 섹터의 대표 고유동성 종목을 web_search로 조회한다.

### 변동성 사전 필터 (선택)

| 변동성 수준 | 기준 (일평균 변동폭 / 주가 비율) |
|------------|--------------------------------|
| 안정형 | 1% 미만 |
| 보통 | 1~3% |
| 공격형 | 3% 이상 |

---

## 1단계 — 유동성 체크

> 가장 중요한 필터. 이걸 통과 못하면 나머지 단계 불필요.

- [ ] 최근 5거래일 일일 거래량 수집
- [ ] 주 평균 거래량 계산
- [ ] 비교 대상 종목(동일 섹터 또는 유사 가격대)과 거래량 비교
- [ ] **판정 기준**: 비교 종목 대비 거래량이 현저히 낮으면 탈락

---

## 2단계 — 변동성 측정

> 매수가/매도가 설정의 기준이 되는 숫자.

- [ ] 일봉 캔들차트에서 최근 20거래일 꼬리(위+아래) 길이 수집
- [ ] 유난히 길었던 날(어닝, 뉴스 이슈) 제외
- [ ] 나머지 평균값 계산 → **하루 평균 변동폭(달러)**
- [ ] 이상치 비율 확인 (전체의 20% 이상이면 변동성 불안정 주의)

---

## 3단계 — 적정가 탐색

> 1주 먼저 사고 한 달 관찰하는 것이 원칙.

- [ ] 현재가가 52주 레인지의 어느 위치인지 확인 (상단 20% / 중간 / 하단 20%)
- [ ] 최근 한 달 평균가 대비 현재가 위치 확인
- [ ] 2단계의 변동폭 x 2 = 단기 적정 진입 여유폭으로 설정
- [ ] **적정가 = 현재가가 최근 평균보다 변동폭 1~2배 이상 눌려있을 때**

---

## 4단계 — 매수가 / 매도가 설정

### 매수가 설정
- [ ] 적정가 기준으로 변동폭 x 1.5~2배 아래를 매수 목표가로 설정
- [ ] "이틀 연속 하락"을 기준으로 삼는 것이 원칙
- [ ] 매수 수량(투자금액) 결정

### 매도가 설정
- [ ] 목표 수익률 결정 (예: 4%, 10% 등 본인 원칙)
- [ ] 매수가 x (1 + 목표수익률) = 세전 매도가
- [ ] 수수료(매수+매도) 차감 후 실질 매도가 확정

---

## 5단계 — 하락장 대응

> 보유 중 급락 상황 발생 시 체크리스트.

- [ ] 기존 보유 물량 → 그대로 유지 (손절 없음 원칙)
- [ ] 평균단가 앱에서 숨기거나 보지 않기
- [ ] 현재 하락한 가격 기준으로 2단계부터 다시 실행
- [ ] 새로운 매수가 / 매도가 설정 → 주문 걸어두기
- [ ] 하루 1회만 확인, 커뮤니티/유튜브 정보 차단

**판단 원칙:**
- 유동성이 갑자기 급감했는가? → 종목 자체 이슈 가능성, 재평가 필요
- 유동성은 유지되는가? → 시장 전체 하락, 원칙대로 버티기

---

## 주의사항

> 이 스킬 사용 시 항상 아래 내용을 출력 마지막에 포함한다.

- 이 방법론은 **손절이 없는 전략**이다. 유동성 필터를 엄격하게 적용하는 것이 유일한 리스크 방어선.
- 기업 분석을 포함하지 않으므로, 펀더멘탈이 붕괴되는 종목(상장폐지 위험, 대규모 사기 등)에는 적용하지 않는다.
- 투자 판단의 최종 책임은 본인에게 있다.
