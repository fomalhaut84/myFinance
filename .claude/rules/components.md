# 컴포넌트 규칙

이 규칙은 src/components/**/*.tsx 파일에 적용.

- 함수형 컴포넌트 + hooks만 사용. class 컴포넌트 금지.
- default export 사용.
- UI 텍스트는 한국어. props/변수명은 영어.
- 금액 표시: 원화는 `toLocaleString('ko-KR')` + "원", 달러는 `$` + 소수점 2자리.
- 수익률: 양수 초록(+), 음수 빨강(-), 소수점 1자리.
- 차트 컬러: 세진=#34d399, 소담=#60a5fa, 다솜=#fb923c 고정.
- 세금 관련 UI에는 항상 "참고용이며 법적 조언이 아닙니다" 면책 문구 포함.
